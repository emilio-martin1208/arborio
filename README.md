# Arborio

A tile-based farming RPG built with [pygame](https://www.pygame.org/). Plant seeds, harvest crops, level up, and unlock new tools and seeds as you grow your farm.

## Running it

```bash
pip install pygame
python3 farm.py
```

## Controls

| Key | Action |
| --- | --- |
| Arrow keys / WASD | Move |
| Space | Plant seed |
| E | Harvest crop / sleep (on the bed tile) |
| Q | Toggle inventory |
| Z | Switch equipped tool |
| R T Y U I O P | Select seed |
| Mouse wheel | Zoom in/out |

## Status

Early WIP — core loop (planting, growing, harvesting, leveling up, tool/seed unlocks) works. More crops, animals, and save/load support are planned.
